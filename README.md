# EMS_WEB_APP
the same employee management system but as an website 

## Vercel Backend

The repository root includes `vercel.json`, `api/index.py`, and `requirements.txt` so a backend Vercel project can deploy from the repo root.

Use these backend project settings:

```text
Framework Preset: FastAPI
Root Directory: ./
Build Command: leave empty
Output Directory: leave empty
```

Set the backend environment variables from `backend/.env.production.example`.
